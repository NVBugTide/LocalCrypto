Encrypt and decrypt files locally without network and server

LocalVault is a small client-side offline tool for encrypting and decrypting files on your own machine. Drop a file, choose the password, and download an encrypted file or a self-decrypting HTML to decript in local browser.

Start encrypting files
AES-256-GCM • PBKDF2-HMAC-SHA-256 key derivation • SELF-DECRYPTING HTML FILE

Why use LocalVault?

Security

Using strong encryption technique that is run via Web Crypto API. No backend database or server data

Creative Outputs

Export either a compact encrypted file or a self-decrypting HTML. The HTML file can be opened in a modern browser by someone who only has the password.

Transparency

The code is plain HTML, CSS and JavaScript. You can inspect it or adapt it for specific purposes.

Password Difficulty

A password-strength meter and adjustable key-derivation settings help you pick a strong password.

INSTRUCTION

  1. Select or drop the file you want to protect.
  2. Choose a password and select a PBKDF2 strength.
  3. Click Encrypt to file or Create self-decrypting HTML.
  4. Remember the password for decryption.
    (Warning: If you lose password, there is no recovery mechanism)

About this project

LocalVault is a small, browser-based file encryption tool built for files security and personal privacy, as a full desktop application would be unnecessary. It is a student-developed study tool.
